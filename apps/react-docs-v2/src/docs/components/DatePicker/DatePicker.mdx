# DatePicker

The DatePicker component allows users to select a date from a calendar view. It provides a user-friendly interface for inputting date values.

## Introduction

DatePicker simplifies date selection with a visual calendar interface. It supports controlled updates, customizable labels, and ensures valid date input.

### Basic Example

import { DatePicker } from '@packages/react-components';

}

## Component Properties

### Controlled State

Use the `value` and `onValueChange` props to control the date picker state. The `value` should be a date string (ISO format is recommended).

<CodeBlock code={`
import { useState } from 'react';
import { DatePicker } from '@packages/react-components';

export const ControlledDatePicker = () => {
	const [date, setDate] = useState<string>('2024-01-01');

    return (
    	<div className="flex flex-col gap-2">
    		<DatePicker
    			value={date}
    			onValueChange={(val) => setDate(val || '')}
    			label="Controlled Date"
    		/>
    		<p>Selected Date: {date}</p>
    	</div>
    );

};
`} lang="tsx" shouldFormat/>

<ControlledDatePicker client:visible />

## API Reference

### DatePicker

#### Props

| Prop            | Type                                         | Default     | Description                                       |
| --------------- | -------------------------------------------- | ----------- | ------------------------------------------------- |
| `label`         | `string`                                     | `undefined` | Label above the input field.                      |
| `value`         | `string`                                     | `undefined` | The selected date value (ISO string recommended). |
| `onValueChange` | `(value?: string, date?: DateValue) => void` | `undefined` | Callback fired when the selected date changes.    |
| `open`          | `boolean`                                    | `undefined` | Whether the calendar popover is open.             |
| `onOpenChange`  | `(details: OpenChangeDetails) => void`       | `undefined` | Callback fired when the open state changes.       |
| `aria-label`    | `string`                                     | `undefined` | ARIA label for accessibility.                     |
| `id`            | `string`                                     | `undefined` | Unique identifier for the component.              |

## Accessibility

- **Keyboard Navigation**: Supports arrow keys for navigating the calendar grid.
- **ARIA**: Includes proper roles and labels for screen readers.
- **Focus Management**: Manages focus between input and calendar popup.
